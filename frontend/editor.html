Параметры показа

Используется только для режима «Жёсткий интервал».
Экстренные слайды показываются по кругу с одинаковой длительностью и не используют обычную настройку частоты.
Изменения применяются к открытому слайду локально. На сервер они попадут только после нажатия основной кнопки «Сохранить».
